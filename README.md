## 🌍 What is Vikarma?

Vikarma means "action beyond karma" — doing what must be done, without attachment to the result.

This application is exactly that: AI power in the hands of every human being, regardless of wealth, geography, or technical skill.

- 🤖 **Multi-AI** — Claude, Gemini, Qwen, Grok, DeepSeek, local Ollama models

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/valentinuuiuiu/vikarma.git
cd vikarma

# Install
npm install
pip install fastmcp httpx anthropic -r server/requirements.txt

# Configure
cp .env.example .env
# Add your API keys (or use free local models)

# Run
npm run electron:dev
```
