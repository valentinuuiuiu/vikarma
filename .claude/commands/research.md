Conduct deep research on: $ARGUMENTS

Follow the `deep-research` skill workflow:

1. Decompose the topic into 3-5 sub-questions
2. Search using multiple Bhairava temples:
   - `wikipedia` for factual background
   - `arxiv` for academic/research angle
   - `duckduckgo` for broad web coverage
   - `news_api` for recent developments
3. Synthesize findings using the `anthropic` temple
4. Store key findings in KAN memory using `remember` tool
5. Output a cited research report with: key findings, evidence quality, contrarian view, and recommendation

Use the output format defined in `skills/deep-research/SKILL.md`.
